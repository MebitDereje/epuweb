# Ethiopian Police University Website

A modern, responsive website for the Ethiopian Police University built with HTML5, CSS3, and vanilla JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Professional design with EPU branding colors (deep blue and gold)
- **Interactive Elements**:
  - Mobile hamburger navigation menu
  - Hero image carousel/slideshow
  - Dynamic news loading
  - Contact form with validation
  - Scroll-to-top button
  - Smooth scrolling

## Pages

1. **Homepage** (`index.html`) - Welcome page with hero slider, quick links, and latest news
2. **About EPU** (`about.html`) - History, mission, vision, and leadership
3. **Academics** (`academics.html`) - Academic programs and departments
4. **Admissions** (`admissions.html`) - Requirements, process, and important dates
5. **Campus Life** (`campus-life.html`) - Facilities, services, and student activities
6. **News & Events** (`news.html`) - University announcements and events
7. **Contact** (`contact.html`) - Contact information and form

## Setup Instructions

1. **Add Images**: Place your images in the `images/` folder:
   - `epu-logo.png` - University logo
   - `campus.jpg` - Campus photo
   - `history.jpg` - Historical photo
   - `president.jpg`, `vp-academic.jpg`, etc. - Leadership photos
   - `news1.jpg` through `news6.jpg` - News images
   - Background images for hero section

2. **Customize Content**: Update the placeholder content in HTML files with actual university information

3. **Update Contact Information**: Replace placeholder phone numbers and email addresses in all pages

4. **Test Responsiveness**: Open the website in different browsers and devices to ensure proper display
